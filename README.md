# svrn_alpha — Website

Corporate website for svrn_alpha. Built with React, TypeScript, Tailwind CSS, and Vite.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Run the dev server:
   ```
   npm run dev
   ```

## Build

```
npm run build
```

## Deployment

Deployed via Vercel from the `main` branch.
