Klare Antworten.

Kann ich jederzeit kündigen?
Ja. Monatsabos enden sofort. Jahresabos enden zur Verlängerung — keine automatische Weiterführung.
Muss ich die Signale selbst handeln?
Nein. Viele Abonnenten lesen das Briefing und beobachten das Desk, ohne auszuführen. Das ist eine gültige Nutzung. Aber alles ist so geschrieben, dass Sie, wenn Sie wollten, handeln könnten.
Ist das Anlageberatung?
Nein. Premise veröffentlicht strukturiertes Research. Das Haus handelt zufällig danach; Ihr Kapital ist Ihres. Jedes Briefing trägt einen klar formulierten Disclaimer und eine Risikoeinordnung.
Was beinhaltet der 7-tägige Test?
Voller Zugang zu allem im Paket — Briefings, Desk, Chronik, Screener. Keine Karte erforderlich. Wir bitten am achten Tag um Zahlung, nicht früher.
Kann ich später das Paket wechseln?
Ja, sofort. Upgrades werden anteilig verrechnet. Downgrades greifen zur nächsten Verlängerung.
